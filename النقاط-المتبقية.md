## ❌ نقاط مفتوحة (تُضاف هنا فور اكتشافها)

### 3. ترجمة الإشعارات حسب لغة المستقبِل ⚠️ يتطلب خطوة يدوية في Supabase
الكود جاهز — الإشعارات تُرسَل بلغة المستقبل. لكن تحتاج إضافة عمود `lang` في Supabase يدوياً:
```sql
ALTER TABLE customers ADD COLUMN IF NOT EXISTS lang text DEFAULT 'ar';
ALTER TABLE salons    ADD COLUMN IF NOT EXISTS lang text DEFAULT 'ar';
```
بعد تشغيل هذا الـ SQL في Supabase Dashboard → SQL Editor، يعمل كل شيء تلقائياً.
