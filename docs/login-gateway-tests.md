# اختبارات بوابات الدخول — Dork App

> يُنفَّذ هذا الاختبار يدوياً بعد كل deploy رئيسي.
> **المطلوب:** متصفح + رقم جوال حقيقي للـ OTP.

---

## 1. بوابة دخول العميل (نقطة 19)

### اختبار OTP بالجوال
- [ ] افتح التطبيق → "سجّل الدخول" → تبويب "الجوال"
- [ ] أدخل رقم جوال سعودي صحيح → اضغط "إرسال رمز"
- [ ] تأكد وصول SMS خلال 30 ثانية
- [ ] أدخل الرمز الصحيح → يجب الدخول مباشرة
- [ ] تأكد ظهور اسم العميل في الـ Drawer

### اختبار PIN
- [ ] بعد الدخول بـ OTP → اضبط PIN من الإعدادات
- [ ] اخرج من الحساب
- [ ] الدخول بتبويب "PIN" → أدخل الرقم → يجب الدخول

### اختبار فشل OTP (نقطة 68)
- [ ] أدخل رمز OTP خاطئ 5 مرات
- [ ] يجب ظهور رسالة قفل مؤقت مع الوقت المتبقي
- [ ] انتظر انتهاء القفل → يعود للعمل

---

## 2. بوابة دخول الصالون (نقطة 18)

### اختبار PIN الصالون
- [ ] افتح التطبيق → "دخول الصالون"
- [ ] أدخل رقم الجوال المسجّل لصالون موجود
- [ ] أدخل PIN الصحيح (6 أرقام) → يجب الدخول للوحة التحكم
- [ ] تأكد ظهور اسم الصالون في الهيدر

### اختبار الحجب بعد 5 محاولات خاطئة
- [ ] أدخل PIN خاطئ 5 مرات متتالية
- [ ] يجب ظهور رسالة "الحساب مقفل مؤقتاً X دقيقة"
- [ ] بعد انتهاء القفل → يعود للعمل

### اختبار استعادة PIN
- [ ] اضغط "نسيت الرمز السري؟"
- [ ] أدخل رقم الجوال + البريد الإلكتروني
- [ ] تأكد وصول OTP على البريد
- [ ] أدخل PIN جديد → يجب الحفظ والدخول

---

## 3. بوابة دخول الأدمن (نقطة 17)

### اختبار كلمة المرور
- [ ] افتح `dork-app.vercel.app/admin` أو الرابط المناسب
- [ ] أدخل بريد الأدمن + كلمة المرور
- [ ] يجب الوصول للوحة الإدارة
- [ ] تأكد ظهور قائمة الصالونات والحجوزات

### اختبار الحماية
- [ ] حاول الوصول لصفحة admin بدون دخول → يجب Redirect لصفحة Login
- [ ] تسجيل الخروج → يجب إلغاء الجلسة وReirect
